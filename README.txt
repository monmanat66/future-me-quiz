# Future Me — Quiz (Static Site)

เปิดไฟล์ `index.html` แล้วเล่นได้ทันที (รองรับมือถือ/เดสก์ท็อป)

## โครงสร้าง
```
future-me-quiz/
  index.html
  assets/
    css/style.css
    js/app.js
    images/ (ผลลัพธ์ 12 รูป + logo_slowcombo.svg)
```

## วิธีใช้งาน
1. เปิด `index.html` ในเบราว์เซอร์
2. กด **เริ่มทำแบบทดสอบ** → ตอบครบ 10 ข้อ
3. ระบบจะคำนวณ Type (A/B/C/D) จาก Q4–Q10 แล้วสุ่มภาพตามคะแนน
4. ปุ่ม **แชร์ผลลัพธ์** ใช้ Web Share API (ถ้าไม่รองรับจะคัดลอกข้อความอัตโนมัติ)
